# Arriva Shop KZ v4.0 - Telegram Bot

Премиум магазин одноразовых вейпов в Казахстане 🔥💨

## Особенности

- 🔥 Каталог из 12 премиум вейпов с автозагрузкой
- 🛒 Корзина и оформление заказа
- 💳 Оплата через Kaspi с автоматическим подтверждением
- 💵 Встречи для оплаты наличными
- 💎 Реферальная программа (500₸ за друга)
- 📊 Полная админ панель с аналитикой
- 📢 Система рассылок с шаблонами
- 🚚 Доставка по Казахстану (Алматы, Астана, Шымкент, Караганда)
- 🎯 Brand voice во всех сообщениях (братан, погнали, огонь!)

## Установка локально

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте `.env` файл:
```bash
cp .env.example .env
```

3. Заполните `.env`:
```
BOT_TOKEN=your_bot_token_from_botfather
DATABASE_PATH=data/bot_data.db
ADMIN_IDS=123456789,987654321
```

4. Запустите бота:
```bash
python bot.py
```

База данных SQLite создастся автоматически в папке `data/` при первом запуске.

## Деплой на BotHosting.ru

1. Загрузите все файлы проекта на BotHosting.ru
2. В настройках бота добавьте переменные окружения:
   - `BOT_TOKEN` - токен вашего бота от @BotFather
   - `ADMIN_IDS` - ID админов через запятую (например: `123456789,987654321`)
3. Запустите бота через панель управления

База данных SQLite будет создана автоматически в папке `data/` (постоянное хранилище на BotHosting.ru).

### Как узнать свой Telegram ID?

Напишите боту @userinfobot - он покажет ваш ID.

## Команды

### Для клиентов:
- `/start` - Главное меню и регистрация
- `/referral` - Реферальная программа (получить ссылку)
- `/help` - Помощь и информация

### Для админов:
- `/admin` - Админ панель с дашбордом
- `/orders` - Список всех заказов
- `/kaspi_paid ORD-...` - Подтвердить оплату Kaspi
- `/meeting_done ORD-...` - Завершить встречу
- `/addproduct` - Добавить новый товар
- `/products` - Список товаров (в разработке)
- `/lowstock` - Товары с низким остатком
- `/stats` - Детальная статистика
- `/top` - Топ-10 товаров
- `/meetings` - Запланированные встречи
- `/referrals` - Статистика рефералов
- `/broadcast` - Рассылка сообщений
- `/settings` - Настройки магазина

├── middleware/         # Middleware
