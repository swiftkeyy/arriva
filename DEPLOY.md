# Инструкция по деплою Arriva Shop KZ v4.0

## Деплой на GitHub

1. **Создай репозиторий на GitHub:**
   - Зайди на https://github.com/new
   - Название: `arriva`
   - Описание: `Arriva Shop KZ v4.0 - Premium Vape Shop Telegram Bot`
   - Сделай приватным (если нужно)

2. **Залей код на GitHub:**

```bash
cd ПРОЕКТЫ/arriva-vape

# Добавь все файлы
git add .

# Сделай первый коммит
git commit -m "Initial commit: Arriva Shop KZ v4.0"

# Добавь remote
git remote add origin https://github.com/swiftkeyy/arriva.git

# Залей на GitHub
git push -u origin main
```

Если будет ошибка с веткой, используй:
```bash
git branch -M main
git push -u origin main
```

## Деплой на BotHosting.ru

### Вариант 1: Через GitHub (рекомендуется)

1. Зайди на BotHosting.ru
2. Создай новый бот
3. Выбери "Deploy from GitHub"
4. Подключи репозиторий `swiftkeyy/arriva`
5. Добавь переменные окружения:
   - `BOT_TOKEN` - твой токен от @BotFather
   - `ADMIN_IDS` - твой Telegram ID (узнай через @userinfobot)
6. Нажми "Deploy"

### Вариант 2: Через загрузку файлов

1. Скачай весь проект как ZIP
2. Загрузи на BotHosting.ru
3. Добавь переменные окружения (см. выше)
4. Запусти бота

## Переменные окружения

Обязательные:
- `BOT_TOKEN` - токен бота от @BotFather
- `ADMIN_IDS` - ID админов через запятую (например: `123456789,987654321`)

Опциональные:
- `DATABASE_PATH` - путь к базе данных (по умолчанию: `data/bot_data.db`)

## Как получить BOT_TOKEN

1. Напиши @BotFather в Telegram
2. Отправь команду `/newbot`
3. Следуй инструкциям
4. Скопируй токен (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Как узнать свой Telegram ID

1. Напиши боту @userinfobot
2. Он покажет твой ID (например: `123456789`)
3. Если админов несколько, укажи через запятую: `123456789,987654321`

## Проверка работы

После деплоя:

1. Найди своего бота в Telegram
2. Отправь `/start` - должно появиться приветствие
3. Отправь `/admin` - должна открыться админ панель
4. Проверь каталог - должно быть 12 товаров

## Troubleshooting

### Бот не отвечает
- Проверь, что BOT_TOKEN правильный
- Проверь логи на BotHosting.ru

### Нет доступа к админ командам
- Проверь, что твой ID в ADMIN_IDS
- Узнай свой ID через @userinfobot

### База данных не создается
- Убедись, что папка `data/` существует
- На BotHosting.ru она создастся автоматически

### Ошибки при запуске
- Проверь, что все зависимости установлены
- Проверь логи: `python bot.py` локально

## Обновление бота

Если используешь GitHub:
1. Внеси изменения в код
2. Закоммить: `git add . && git commit -m "Update"`
3. Залей: `git push`
4. На BotHosting.ru нажми "Redeploy"

## Бэкап базы данных

База данных находится в `data/bot_data.db`

Для бэкапа:
1. Скачай файл `data/bot_data.db` с сервера
2. Сохрани локально
3. При необходимости восстанови, загрузив обратно
